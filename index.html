How to Add New Recipes

To add a new recipe:

  1. Make a copy of the recipe-template.html file.
  2. Rename the copied file to something descriptive (e.g., chocolate-cake.html). Make sure it ends with .html.
  3. Open the new HTML file in a text editor.
  4. Replace all the placeholder text (like "[Recipe Title Placeholder]", "[Ingredient 1 Placeholder with quantity]", etc.) with the actual details for your recipe.
  5. Save the changes to your new recipe file.
  6. Open this index.html file in a text editor.
  7. Add a new line in the "Available Recipes" list above, like this:
    <li><a href="your-new-recipe-file.html">Name of Your Recipe</a></li>
    (Replace your-new-recipe-file.html with the actual filename and "Name of Your Recipe" with its title).
  8. Save this index.html file.

Now, when you open index.html in a web browser, you should see a link to your new recipe!
